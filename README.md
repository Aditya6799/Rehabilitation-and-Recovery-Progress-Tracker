# Rehabilitation and Recovery Progress Tracker

A premium rehabilitation and recovery progress tracking platform for patients and clinicians.

## Features
- **Dashboard**: High-level overview of recovery progress.
- **Progress Tracking**: Log sessions and visualize recovery data with interactive charts.
- **Exercise Management**: Access and manage personalized exercise routines.
- **Appointments**: Schedule and track consultations with healthcare providers.
- **AI-Powered Insights**: Get recovery suggestions and guidance.

## Tech Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript.
- **Backend**: Node.js, Express.
- **Database**: MongoDB.
- **AI**: Google Gemini.

## Getting Started
1. Clone the repository.
2. Navigate to the `backend` directory.
3. Install dependencies: `npm install`.
4. Set up your `.env` file with necessary credentials.
5. Start the server: `npm start` or `node server.js`.
6. Open the frontend in your browser.
